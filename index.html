Recents projects

AxonDeepSeg (2017)

AxonDeepSeg is a segmentation software for microscopy data of nerve fibers. It is a CNN using a variation of the U-net architecture to segment microstructures (axon and myelin) from electron microscopy data from the spinal cord. AxonDeepSeg is released as an open-source software (Python, TensorFlow) available online on [GitHub].
The project is the result of the collaboration of several people. It is lead by the neuroimagery laboratory [NeuroPoly], from Polytechnique Montréal. I took over the project in May 2017 and worked on it until September 2017 as part of a summer internship. The documentation can be found [here].
I am also one of the two lead authors of a scientific article on this software and the structure we developed. The pre-print is available on [arXiv].

Atrous convolutions (2018)

I worked on the use of atrous convolutions in semantic segmentation as part of my class on Object detection and Computer Vision from the master MVA. I leant on the code from my summer open-source project, AxonDeepSeg, and explored the use of atrous convolutions following the conclusions from the [DeepLab] paper through qualitative and quantitative analysis.
I also implemented Atrous Spatial Pyramid Pooling from DeepLab, tried different parameterizations and architectures, and compared the best results with the baseline, AxonDeepSeg.

Graph exploration with visibility constraints using bandits (2018)

I worked on a stochastic graph exploration project as a part of my class on Graphs in Machine Learning for the Master MVA. With my colleague, we implemented the deterministic NetExp algorithm [Singla15] for graph exploration under limited visibility constraints. We then derived a new algorithm in the stochastic paradigm, taking ideas from the Full information bandit and the Side-observation bandit settings, and leaning on previous work on the stochastic graph exploration problem. We used an alternative to Thompson Sampling to tackle the exploration/exploitation bias and obtained better results than the other methods.
The implementation can't be shared right now, but a readme file is available [here].
